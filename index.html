Unidad 1

Introducción a Linux y la Consola Bash

En esta unidad conocerás los fundamentos de Linux, su historia, las distribuciones, la terminal y el funcionamiento de Bash.

Historia de Linux

Linux nació en 1991 gracias a Linus Torvalds y actualmente es uno de los sistemas operativos más utilizados en servidores, supercomputadoras y dispositivos móviles.

Linux es software libre y cualquier persona puede modificar su código fuente.

¿Qué es un Sistema Operativo?

Es el software encargado de administrar el hardware y permitir la ejecución de programas.

Windows, Linux y macOS son sistemas operativos.

Características de Linux

  • Libre
  • Seguro
  • Multitarea
  • Multiusuario
  • Estable
  • Personalizable

Distribuciones Linux

Ubuntu Debian Fedora Arch Linux Kali Linux Linux Mint

Terminal Linux

La terminal permite controlar completamente el sistema mediante comandos.

¿Qué es Bash?

Bash significa Bourne Again Shell y es el intérprete de comandos utilizado en la mayoría de distribuciones Linux.

Unidad 2

Comandos Básicos de Bash

Aprende los principales comandos utilizados diariamente en Linux.

pwd

Muestra la ruta del directorio actual.

pwd

ls

Lista el contenido de una carpeta.

ls

cd

Cambia de directorio.

cd Documentos

mkdir

Crea una carpeta nueva.

mkdir Proyecto

touch

Crea un archivo vacío.

touch archivo.txt

cp

Copia archivos o carpetas.

cp origen destino

mv

Mueve o renombra archivos.

mv archivo.txt carpeta/

rm

Elimina archivos.

rm archivo.txt

cat

Muestra el contenido de un archivo.

cat archivo.txt

grep

Busca texto dentro de archivos.

grep "Linux" notas.txt

find

Busca archivos y carpetas.

find /home -name archivo.txt

history

Muestra el historial de comandos.

history

Usuarios

Grupos

ps

top

kill

Unidad 4

Automatización y Scripts Bash

¿Qué es un Script?

Un script Bash es un archivo que contiene una serie de comandos que pueden ejecutarse automáticamente.

Estructura de un Script


#!/bin/bash

echo "Hola Mundo"

Variables


nombre="Mateo"

echo $nombre

Condicional IF


if [ $edad -ge 18 ]

then

echo Adulto

fi

Bucle FOR


for i in 1 2 3

do

echo $i

done

Funciones


saludar(){

echo Hola

}

Cron

Cron permite programar tareas automáticas.
